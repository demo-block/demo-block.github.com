(1)塔罗币概述

核心原理是区块链账本本身被称为区块链，塔罗币是指将新的交易记录添加 到以太坊分布式公共账本并赚取通证的过程。

(2)工作量证明(PoW)

指将一组交易记录添加到分布式网络所需要满足的条件。每组交易被打包压 缩在区块头重，区块头还包含一段随机字符(NOUNCE)， 矿机需要搜寻计算正确 的字符串使得区块满足一定的条件。对以太坊而言，则需要满足区块的 SHA-256 哈希值。

当区块链选择 PoW(Power-of-Work)作为共识机制时，每一次区块的生 成消耗的算力都将成为其价值的基石。另外，链条上的每个节点都具备解决现实 环境问题的能力，并能对外提供各种产品与服务。如果每个节点能够参与共享工 作的结算，整个区块链条就具备了现实的产出价值。PoW 的核心要义为:算力 越大，挖到块的概率越大，维护区块链安全的权重越大。

(3)设计原理

确保了以太坊以稳定的速率分发，同时激励节点参与维护系统的安全。

(4)难度调节

改变 SHA-256 哈希值前导 0 的个数，尝试找到以很多 0 开始的随机散列。

3)以太坊塔罗币:哈希运算方法

哈希运算是指输入任意长度的字符串而得到给定长度的字符串的过程。

哈希 运算法主要包括以下三种:SHA-256 哈希运算输入所有的交易信息并得到一个 固定的 256 位的字符串，这是以太坊最原始的哈希算法;

Scrypt 是一种简化的 算法，更容易在 CPU 上进行计算，并且比 SHA-256 消耗低，所以更受青睐; X11 是一种较为安全的哈希算法，GPU 和 CPU 都可以被用于计算，由于处理能 力要求较低，也能降低 GPU 的温度，降低电力成本，所以目前被几种加密货币 使用，包括 Dash。
